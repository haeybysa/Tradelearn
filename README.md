# Tradelearn
TradeLearn — бесплатный курс трейдинга с нуля для начинающих инвесторов. 8 интерактивных уроков, практический симулятор торговли с виртуальным портфелем 100 000 ₽, изучение технических индикаторов (RSI, MACD, скользящие средние) и правил управления рисками. Учитесь безопасно — без реальных денег, но с реальными знаниями.
